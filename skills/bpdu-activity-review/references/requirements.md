# Activity Review Requirements Reference

## Source: Club Maintenance.md — Club Activity Review

**Original Notice:**

通知：
各位社长们：
上午好！请汇总本周开展社团活动的剪影。活动剪影需上交2-3张本周开展的活动照片（要求图像清晰可辨别，有三分之二参与活动的社员出镜），并附上以第三人称编辑50-100字的文案。以WORD的形式上交，命名为"国际学院XX社团第七周活动剪影" 发送至各社团对接的干事， 上交截止时间为11月2日（本周日)22:00前。

## Exact Standards

### Photos
- Quantity: 2–3 per event (minimum 2, maximum 3)
- Quality: Clear, identifiable faces
- Composition: At least 2/3 of participating members visible
- Must be actual activity photos (not screenshots or unrelated images)

### Description
- Length: 50–100 words (WPS word count)
  - Chinese characters: each counts as 1 word
  - English words (whitespace-separated): each counts as 1 word
- Language: Predominantly Chinese (>50% Chinese characters); English allowed for names, motions, debate technique names
- Time reference: Must use last-week format (e.g., 上周三, 上周五), matched to actual event date from event-preview schedule
- Perspective: Third-person (第三人称) — no 我们/我社/笔者/本人
- Tone: Professional, descriptive

### Output File
- Format: Word document (.docx)
- Naming: `国际学院BP_Debate_Union_第X周活动剪影.docx`
- Output path: `skills/bpdu-activity-review/output/`

### Submission
- Deadline: Sunday 22:00 each week
- To: Assigned student affairs 干事 (varies by week — check weekly notification)
- Format: Word document with photos embedded
