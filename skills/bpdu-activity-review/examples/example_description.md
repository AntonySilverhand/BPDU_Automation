# Activity Review Example

## Good Example Description (50-100 words, third-person, Chinese)

```
11月12日，BP Debate Union在博闻楼B-606开展了苏格拉底式研讨会活动。本次活动共有20名同学参加，围绕社交媒体的continuing the spark现象展开深入讨论。同学们积极发言，热烈探讨，不仅提升了批判性思维和辩证能力，也增强了表达能力和团队协作意识。大家表示此类活动非常有意义，期待下次继续参与。
```

Word count: ~80 words (Chinese)
Time reference: 11月12日 — note: must ALSO include last-week format like 上周三 in the text.

## Photo Requirements

- 2-3 photos per event
- Minimum 2, maximum 3
- Clear, identifiable
- 2/3 of participants visible

## Output File

`国际学院BP_Debate_Union_第7周活动剪影.docx`
